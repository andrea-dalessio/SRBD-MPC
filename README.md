# Single Rigid Body Dynamics - MPC  

## Introduction  

This project aims to implement the SRBD-MPC as proposed by the paper **"A Real-Time Approach for Humanoid Robot Walking including Dynamic Obstacles Avoidance"** starting from the IS-MPC controller proposed by Prof. Nicola Scianca ([Github link](https://github.com/DIAG-Robotics-Lab/ismpc.git)), excluding the obstacle avoidance section and implementing a robust disturbance recovery system.
